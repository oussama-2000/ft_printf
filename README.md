#  ft_printf

A custom implementation of the standard C `printf` function — part of the 42 cursus.

---

##  Project Overview

The goal of **ft_printf** is to recreate the behavior of the C standard library function `printf()`.  
This project deepens your understanding of **variadic functions**, **macros**, **data formatting**, and **low-level memory and I/O management** in C.
